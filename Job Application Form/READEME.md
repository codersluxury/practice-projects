# Job Application Form


A responsive and modern Job Application Form built using HTML5 and CSS3. This project allows users to submit their personal details, preferred job position, availability, and a short message explaining why they are a great fit for the role.


## Features


• Clean and modern UI design

• Fully responsive layout for desktop and mobile devices

• Form validation using HTML required attributes

• Custom styled radio buttons

• Dropdown selection for job positions

• Textarea for applicant message

• Interactive hover and focus effects


## Technologies Used


• HTML5 – Structure of the form

• CSS3 – Styling and responsiveness


## Project Structure


project-folder/

│

├── index.html

└── styles.css


## Form Fields


The application form includes:

• Full Name – Text input

• Email Address – Email input validation

• Position – Dropdown menu with options:

• Developer

• Designer

• Manager

• Availability – Radio button options:

• Full-Time

• Part-Time

• Message – Textarea for applicant introduction

• Submit Button – Sends the application


## Design Highlights


• Uses CSS variables for easy theme customization

• Soft background gradient for a modern look

• Smooth focus animations on form inputs

• Custom radio button styling

• Mobile-friendly responsive design using media queries


## How to Run


1. Download or clone this repository:


git clone https://github.com/your-username/job-application-form.git


2. Navigate to the project folder:


cd job-application-form


3. Open index.html in your browser.


## Customization


You can easily modify:

• Form colors in the :root variables section

• Available job positions in the <select> menu

• Form layout spacing and responsiveness in styles.css

• Button colors and hover effects
