# Business Card

A responsive digital business card created with HTML and CSS, tweaked to work seamlessly on all devices including mobile.

## 📋 Description

This project showcases a clean, professional business card layout featuring:
- Profile image
- Contact information (name, title, company)
- Email and phone details
- Portfolio link
- Social media connections

**Note:** I've customized this practice project by making it fully responsive across all devices, including mobile phones and tablets, ensuring optimal viewing and usability on any screen size.

## 🛠️ Technologies Used

- **HTML5**: Semantic structure and content
- **CSS3**: Styling, layout, and responsive design

## 🚀 How to View

1. Open `index.html` in your web browser
2. The business card will display with all contact information and links
3. Test responsiveness by resizing your browser window or viewing on mobile devices

## 📁 Files

- `index.html` - Main HTML structure with business card content
- `styles.css` - CSS styling for layout, colors, and responsive behavior

## 📱 Responsiveness Features

- Fluid width layout (70% of container)
- Centered design that adapts to different screen sizes
- Mobile-friendly viewport configuration
- Optimized typography and spacing for various devices
