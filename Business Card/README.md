# Business Card

A digital business card created with HTML and CSS.

## 📋 Description

This project showcases a clean, professional business card layout featuring:
- Profile image
- Contact information (name, title, company)
- Email and phone details
- Portfolio link
- Social media connections

## 🛠️ Technologies Used

- **HTML5**: Semantic structure and content
- **CSS3**: Styling and layout

## 🚀 How to View

1. Open `index.html` in your web browser
2. The business card will display with contact information and links
3. The layout is centered and should display well on desktop screens

## 📁 Files

- `index.html` - Main HTML structure with business card content
- `styles.css` - CSS styling for layout and visual design

## 🎨 Design Details

- **Color Scheme**: Rosybrown background with a white card container
- **Card Width**: Fixed width of 300px for consistent presentation
- **Typography**: Arial sans-serif font family, 16px base size
- **External Links**: Links open in new tabs using `target="_blank"`
- **Section Dividers**: Horizontal rules separate contact, portfolio, and social sections
- **Layout**: Centered card with 20px padding and balanced spacing

## 📱 Mobile Notes

- A mobile-friendly viewport meta tag is included in the HTML
- The image scales with the container using `max-width: 100%`
