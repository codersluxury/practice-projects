# Business Card

A simple digital business card created with HTML and CSS.

## 📋 Description

This project showcases a clean business card layout featuring:
- Profile image
- Contact information (name, title, company)
- Email and phone details
- Portfolio link
- Social media connections

## 🛠️ Technologies Used

- **HTML5**: Semantic structure and content
- **CSS3**: Styling and layout

## 🚀 How to View

1. Open `index.html` in your web browser
2. The business card will display with all contact information and links

## 📁 Files

- `index.html` - Main HTML structure
- `styles.css` - CSS styling for the business card layout
