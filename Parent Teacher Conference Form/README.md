# Parent Teacher Conference Form

An interactive parent-teacher conference scheduling form built with HTML and CSS.

## 📋 Description

This project demonstrates form creation and styling with the following features:
- Student information section with name and grade input
- Parent/guardian information section
- Contact method selection (email or phone)
- Additional notes textarea for discussing specific concerns
- Professional, user-friendly form layout with fieldsets and legends

## 🛠️ Technologies Used

- **HTML5**: Semantic form structure with fieldsets, labels, and input types
- **CSS3**: Styling and form layout

## 🚀 How to View

1. Open `index.html` in your web browser
2. The form will display with all input fields and sections
3. Fill out the form fields to see the interactive elements

## 📁 Files

- `index.html` - Main HTML form with student, parent, and contact information fields
- `styles.css` - CSS styling for form layout and visual design

## 🎨 Design Details

- **Form Structure**: Organized fieldsets for logical grouping of related information
- **Input Types**: Text inputs, number input, radio buttons, and textarea
- **Accessibility**: Proper label associations with `for` and `id` attributes
- **Validation**: Required fields marked with the `required` attribute
- **Placeholder Text**: Example values provided for user guidance
- **Submit Button**: Styled submit button for form submission

## 📝 Form Sections

1. **Student Information**: Student name and grade
2. **Parent/Guardian Information**: Parent or guardian name
3. **Preferred Contact Method**: Radio buttons for email or phone selection
4. **Additional Notes**: Text area for specific concerns or discussion topics
