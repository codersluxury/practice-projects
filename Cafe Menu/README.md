# Cafe Menu 🎓

A responsive cafe menu webpage built as a practice project from [FreeCodeCamp](https://www.freecodecamp.org/). This project demonstrates fundamental HTML and CSS concepts including layout, styling, and responsive design.

## 📸 Features

- **Clean Menu Layout**: Coffee and desserts sections with prices
- **Responsive Design**: Centered layout that adapts to different screen sizes
- **Styled Typography**: Custom fonts and text alignment for visual hierarchy
- **Interactive Elements**: Hover effects on links for better user experience
- **Professional Theme**: Warm, cafe-inspired color palette with background imagery

## 🛠️ Technologies Used

- **HTML5**: Semantic markup and structure
- **CSS3**: Layout styling, flexbox principles, and responsive design

## 📁 Project Structure

```
.
├── index.html      # Main HTML file with menu content
├── styles.css      # Stylesheet with all styling
└── README.md       # This file
```

## 🚀 Getting Started

### Prerequisites
No special prerequisites needed! This is a simple static website.

### Running Locally

1. Clone the repository:
```bash
git clone https://github.com/YOUR_USERNAME/cafe-menu.git
cd cafe-menu
```

2. Open `index.html` in your web browser:
   - Double-click `index.html`, or
   - Right-click → Open with → Your preferred browser
